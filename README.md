#  ESP-32-Relay Control System
This project controls a 48V solenoid for the Mesa FSD kart using an **ESP32 WROOM-32**.

## Hardware Setup
* **Microcontroller:** ESP32 WROOM-32
* **Power:** 48V Battery system with a 5V PDB for logic.
* **Switching:** PWM triggered by GPIO 27.

## How to Run
1. Open the `.ino` file in Arduino IDE.
2. Select **DOIT ESP32 DEVKIT V1**.
3. Upload to the board.
